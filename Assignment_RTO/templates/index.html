RTO Managament Application

Register new RTO incharge details Register new User

List RTOs List Users

Web Application

  • Language: Python
  • Framework: Flask

App Routes

  • /RTO/register [GET]: Show new RTO registration form
  • /User/register [GET]: Show new User(Vehicle) registration form
  • /RTO/list [GET]: Show new RTO list
  • /User/list [GET]: Show new User list
  • /RTO/persist [POST]: Persist new RTO details to DB
  • /User/persist [POST]: Persist new User(vehicle) details to DB

Database Design

  • Database: RTO
  • Tables: rto_details, user_registrations

Front end

  • User Interface: HTML5, Bootstrap CSS
  • Server Requests: jQuery ajax requests
